INTRODUCCIÓN

La nivelación de bancos de nivel es un proceso fundamental en la topografía que permite establecer el control vertical de un área de estudio. En este proyecto, se aplican métodos de nivelación geométrica para determinar cotas precisas, esenciales para el diseño y ejecución de obras civiles

OBJETIVO

Establecer y ajustar las elevaciones de distintos Bancos de Nivel (BM) mediante el uso de nivelación diferencial, aplicando el método de mínimos cuadrados para la corrección de errores y garantizando la precisión requerida para el proyecto.

Bancos de nivel

Definición

Un Banco de Nivel (Bench Mark) es un punto de control vertical de carácter permanente o semipermanente, cuya elevación o cota es conocida con precisión respecto a un plano de referencia, generalmente el Nivel Medio del Mar (NMM). Estos puntos constituyen la base de toda la red altimétrica de un proyecto de ingeniería.

Conformación de un BM

Para garantizar que la cota no sufra variaciones por asentamientos o factores externos, los BM se materializan físicamente mediante hitos de hormigón de dimensiones normalizadas o placas de bronce empotradas en afloramientos rocosos o estructuras de concreto de alta estabilidad.

Tipos de nivelación

La nivelación geométrica o diferencial es el método principal utilizado en este proyecto. Se basa en la determinación de la diferencia de altura entre dos puntos mediante lecturas horizontales dirigidas a una mira graduada (estadal). Dependiendo de la extensión, puede ser:

  • Nivelación Simple: Cuando el desnivel se obtiene con una sola estación del instrumento.

  • Nivelación Compuesta: Cuando es necesario realizar múltiples estaciones para unir dos puntos distantes.

Precisión y orden de las nivelaciones

La calidad de una nivelación se mide por su error de cierre. Según la normativa técnica, la precisión se clasifica en órdenes. La tolerancia (\(T\)) permitida para un circuito de tercer orden (común en ingeniería civil) se rige por: \[\begin{equation} T = \pm m \sqrt{K} \end{equation}\] Donde:

  • \(m\): Coeficiente de error (mm).

  • \(K\): Distancia total recorrida en kilómetros.

Error y ajustes

Debido a que toda medición física conlleva errores accidentales, el circuito de nivelación rara vez cierra perfectamente. Si el error de cierre vertical (\(e_v\)) es menor o igual a la tolerancia (\(T\)), se procede al ajuste de las cotas.

Métodos de corrección

Para este proyecto se aplica el rigor estadístico del Método de Mínimos Cuadrados, utilizando el sub-método de Ecuaciones de Condición. Este método garantiza que la suma de los cuadrados de las correcciones ponderadas sea mínima, proporcionando los valores más probables.
